## 트리 알아보기
- 트리의 특징
  - 순환 구조를 지니고 있지 않고, 1개의 루트 노드가 존재
  - 루트 노드를 제외한 노드는 단 1개의 부모 노드를 갖는다.
  - 트리의 부분 트리 역시 모든 특징을 따른다.
