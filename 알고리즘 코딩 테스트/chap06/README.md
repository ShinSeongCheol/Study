# 그리디
- 현재 상태에서 볼 수 있는 선택지 중에 최선의 선택을 하는 알고리즘
- 동적 계획법보다 구현하기 쉽고 시간복잡도가 우수
- 항상 최적의 해를 보장하지 못하다는 단점도 있다.
- 그리디 알고리즘을 적용할때 논리 유무를 충분히 살펴봐야한다.
